Initialization

Launchpad is developed with latest technology.

If you want to modify templates without use of saas you can directly edit into main.css present in css folder.

If you want to modify templates with saas you have to first install npm using npm install command in root HTML folder from command prompt. All the node modules will be downloaded using this and you can do further modification in template with just one command gulp serve and you can edit main.scss and it will be compiled into main.css file present in css folder.

Uility CSS Classes

  • .mx-5, .mx-10,.mx-15 - .mx-50 for margin-right and margin-left
  • .my-5, .my-10,.my-15 - .my-50 for margin-top and margin-bottom
  • .px-5, .px-10,.px-15 - .px-50 for padding-right and padding-left
  • .py-5, .py-10,.py-15 - .py-50 for padding-top and padding-bottom
  • .mt/r/b/l-5, .mt/r/b/l-10, .mt/r/b/l-15 - .mt/r/b/l-50 for margin-top/right/bottom/left
  • .pt/r/b/l-5, .pt/r/b/l-10, .pt/r/b/l-15 - .pt/r/b/l-50 for padding-top/right/bottom/left
  • .font-dosis, .font-roboto, .font-raleway, .font-nunito for font-family
  • .fs-1, .fs-2, .fs-3 - .fs-52 font-size in px
  • .fs--1, .fs--2, .fs--3 - .fs--10 font-size in rem
  • .fw-100, .fw-200, .fw-300 - .fw-600 to change font-weight
  • .lh-5, .lh-10, .lh-15 - .lh-20 line-height in px
  • .lh--1, .lh--2, .lh--3 - .lh--10 line-height in rem
  • .opacity-5, .opacity-10, .opacity-15 - .opacity-100 opacity in %
  • .cursor-pointer, .cursor-crosshair to change mouse cursor
  • .bg-fixed to make a simple parallax effect
  • .hr-gradient-center, hr-gradient-l2r, hr-gradient-r2l make a gradient thin line
  • .gradient-btn-1, .gradient-btn-2, .gradient-btn-3 - .gradient-btn-6 make a gradient button
  • .dark-olay-1, .dark-olay-2, .dark-olay-3 - .dark-olay-9 dark overlay over image
  • .light-olay-1, .light-olay-2, .light-olay-3 - .light-olay-9 light overlay over image
  • .h--10vh, .h--20vh, .h--30vh - .h--100vh viewport height in vh
  • .h--10%, .h--20%, .h--30% - .h--100% viewport height in px
  • .w--10%, .w--20%, .w--30% - .w--100% viewport width in %

Mailgun setup

Mailgun Setup

Easy SMTP integration and a simple, RESTful API abstracts away the messy details of sending transactional or bulk email. Scale quickly, whether you need to send 10 or 10 million emails.

We have used mailgun to send contact form information to the email id specified.

You just need to insert your mailgun API key

If you don't have mailgun API key you can generate from here.

https://www.mailgun.com/

You can find the below code in sendmail.php file

<?php
//composer require mailgun/mailgun-php:~1.7.2
# Include the Autoloader (see "Libraries" for install instructions)
require 'vendor/autoload.php';
use Mailgun\Mailgun;
if (isset($_POST['sname'])) {
$sname=$_POST['sname'];
$email=$_POST['email'];
$subject = $_POST['subject'];
$msg = $_POST['msg'];
$mgClient = new Mailgun('key-yourAPIkey');
//enter domain which you find in Default Password 
$domain = "mg.galacticlab.com";

# Make the call to the client.
$result = $mgClient->sendMessage($domain, array(
    "from" => "$sname <mailgun@mg.galacticlab.com>",
    "to" => "youremail@example.com",
    "subject" => "$subject",
    "html" => "$email<br><br>$msg"
));
echo '<script>
        window.location = "http://themes.galacticlab.com/html-cloud/launchpad/thank-you.html";
    </script>';
}
?>
